# KNN-collection
В данном репозитории представлены четыре работы, посвященные применению метрических алгоритмов машинного обучения для решения задач классификации. Материалы охватывают как базовые подходы к многоклассовой классификации, так и более продвинутые техники ускоренного поиска ближайших соседей.

### 1. Многоклассовая классификация

В этой работе рассматривается задача классификации цветков ириса (датасет Iris) с использованием логистической регрессии. Основное внимание уделяется двум стратегиям решения многоклассовых задач: подход "один против всех" (One-vs-Rest), при котором задача сводится к серии бинарных классификаций, и мультиномиальный подход (Multinomial), обобщающий функцию потерь логистической регрессии на многоклассовый случай. Мультиномиальный подход показал идеальную точность (100%) на тестовой выборке.

### 2. KNN для распознавания рукописных цифр

Здесь реализована классификация рукописных цифр из датасета Digits с помощью алгоритма K ближайших соседей. Изображения размером 8×8 пикселей преобразуются в векторы признаков, после чего исследуется влияние количества соседей и типа весов (равномерные или зависящие от расстояния) на качество классификации. Оптимальная точность составила около 98.7%.

### 3. Быстрый поиск соседей с LSH

Данная работа демонстрирует реализацию приближенного поиска ближайших соседей с использованием метода Locality Sensitive Hashing (LSH). Идея заключается в кодировании объектов с помощью случайных гиперплоскостей: для каждого объекта определяется его положение относительно нескольких плоскостей, что позволяет сформировать битовую маску (хеш). Объекты, попавшие в одну корзину, считаются потенциально похожими, и точный поиск выполняется только среди них. Такой подход значительно ускоряет работу на больших объемах данных и снижает потребление памяти по сравнению с полным перебором.

### 4. Домашнее задание: классификация вин

Финальная практическая работа посвящена классификации вин по их химическим характеристикам (датасет Wine). Задачи включают анализ баланса классов, масштабирование признаков с помощью StandardScaler, обучение базовой модели KNN, подбор гиперпараметров (количество соседей и тип весов) с помощью GridSearchCV, а также визуализацию матрицы ошибок. Особое внимание уделено сравнению точного поиска ближайших соседей с приближенным поиском на основе LSH. Для оценки качества модели используется взвешенный F1-score.
